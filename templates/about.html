About The Project

ResearchPortal is an AI-powered platform that leverages state-of-the-art language models and RAG (Retrieval-Augmented Generation) to analyze research papers, generate detailed summaries, answer contextual questions, and visualize key insights.

AI-Powered Analysis

Advanced NLP and Groq-powered LLMs for accurate section detection, summarization, and semantic understanding.

Vector RAG

Retrieval-Augmented Generation ensures contextually accurate answers using embeddings and vector databases.

Visual Insights

Interactive charts and trend analysis to visualize keywords, topics, and content distribution patterns.

Tech Stack

Flask • LangChain • Groq LLM • HuggingFace Embeddings • Chart.js • RAG Pipeline
